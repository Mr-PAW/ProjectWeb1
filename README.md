# Yogyakarta Tourism Destination Web

Sebuah web aplikasi untuk menampilkan berbagai destinasi wisata yang ada
di Yogyakarta.\
Project ini memiliki fitur autentikasi, sistem komentar, serta manajemen
destinasi wisata dengan role **Admin** dan **User**.

------------------------------------------------------------------------

## Features

### Authentication

-   Login dan Register untuk pengguna baru.
-   Role system terdiri dari:
    -   **Admin**
    -   **User**

------------------------------------------------------------------------

## Role User

### User dapat:

-   Melihat seluruh destinasi wisata.
-   Membaca detail informasi tempat wisata.
-   Menambahkan komentar pada destinasi wisata.
-   Menghapus komentarnya sendiri.

------------------------------------------------------------------------

## Role Admin

### Admin dapat:

-   Melakukan CRUD (Create, Read, Update, Delete) destinasi wisata.
-   Menghapus komentar dari pengguna manapun.

------------------------------------------------------------------------

## Tech Stack (opsional)

-   Frontend: HTML, CSS, CSS Boostrap
-   Backend: PHP 
-   Database: MySQL

------------------------------------------------------------------------

## Installation & Setup (opsional)

1.  Clone repository

2. Masukkan sql ke MySql

3. Jalankan pada server lokal
